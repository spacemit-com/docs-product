# MUSE

![MUSE](./static/muse_front.png)

进迭时空MUSE系列生态产品，基于高性能M1芯片，旨在为所有RISC-V的热爱玩家、创新型开发者、技术爱好者、研究人员等提供更高效便捷的开发平台，竭力打造行业硬件产品标杆，为RISC-V的生态共建提供有力的工具支持。
MUSE系统生态产品序列，涵盖了SOM（核心板）、Pi/Pi Pro（各尺寸开发板）、Box（Mini PC）、Book（笔记本电脑）、Shelf（开发者服务器）等五款常见的硬件产品，让开发者可以自由选择更适合自己开发环境的产品。它提供了一系列针对开发者习惯的功能和资源，帮助开发者更高效地创建、部署和管理他们的项目。同时，产品具有简洁直观的界面，使开发者能够轻松上手并迅速开始工作，支持常规编译工具，并针对编译性能做了优化。
此外，进迭时空还提供了丰富的文档和社区支持，确保开发者在使用过程中遇到的问题可以及时解决。无论是刚刚起步的新手还是经验丰富的专业开发者，都能从中受益。MUSE系列生态产品，希望能全方位帮助个人开发者更快释放创造力，实现技术梦想，为机构开发者打造出标准化硬件方案，整合供应链，帮助产品的高效落地。
让更多的人拥抱RISC-V，一起共建更美好的生态。
